# GY-85

Arduino implementation for GY-85 (ADXL345 accelerometer, ITG3200 gyroscope and HMC5883L magnetometer)

- [HMC5883L magnetometer](http://dlnmh9ip6v2uc.cloudfront.net/datasheets/Sensors/Magneto/HMC5883L-FDS.pdf)

## Wiring

- **[GY-85]** -> **[Arduino]**
- VCC_IN -> 3.3V
- GND -> GND
- SCL -> A5
- SDA -> A4
